## Design Context

### Users
Developers and programming language enthusiasts comparing language performance. They want quick, scannable benchmark results with visual charts and rankings.

### Brand Personality
Technical, precise, fast, no-nonsense.

### Aesthetic Direction
Dark terminal — matches the benchmark chart aesthetic. Dark navy background (#1e1e2e), bright accent colors per language, monospace touches for code/data, clean sans-serif for body. Inspired by code editors and terminal UIs. Anti-reference: generic AI gradient sites.

### Design Principles
1. Data-first: results and charts are the hero, not decorative elements
2. Dark cohesion: match the existing chart color palette exactly
3. Scannable: rankings table + charts visible without scrolling fatigue
4. Language identity: each language has its icon and consistent color
