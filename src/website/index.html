About

How it works

Test files are placed in src/tests/ with the same name across languages (e.g. hello.c, hello.py, hello.vdx). The benchmark script auto-detects all available compilers and interpreters on PATH, compiles C/C++ with -O2, and runs each test 10 times to compute a median.
